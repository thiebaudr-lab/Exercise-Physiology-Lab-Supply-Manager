📋 First-Time Setup Instructions
  1. 1
    Create a Google Sheet
    Go to sheets.google.com and create a new blank spreadsheet.
  2. 2
    Open Apps Script
    In the Sheet, click Extensions → Apps Script. Paste the contents of Code.gs and save.
  3. 3
    Initialize the Sheets
    In the Apps Script editor, select the function initializeSheets from the dropdown and click Run. Authorize when prompted.
  4. 4
    Deploy as Web App
    Click Deploy → New Deployment → Web App. Set Execute as: Me, Who has access: Anyone (or your org). Copy the Web App URL.
  5. 5
    Paste URL into api.js
    Open api.js and replace YOUR_APPS_SCRIPT_URL_HERE with your URL. You're live!
